四、搭建 GitHub 仓库（存储博客静态页面）

  1. 注册 / 登录 GitHub：访问官网 https://github.com/，已有账号直接登录，无账号注册即可（免费）。
  2. 新建仓库：点击右上角「+」号 → 选择「Create a new repository」。
  3. 仓库关键配置（必须按要求填写）：
    • 仓库名：你的GitHub用户名.github.io（例如我的用户名是 xxxx，仓库名就是xxxx.github.io）。
    • 可见性：勾选「Public」（GitHub Pages 仅支持公开仓库，私有仓库无法访问）。
    • 其他选项：勾选「Add a README file」（可选，方便记录仓库说明）。
  4. 完成创建：拉到页面底部，点击「Create repository」即可。

常见问题排查

  1. 执行 hexo 命令提示 “command not found”：前面加 npx（如 npx hexo g）。
  2. 部署失败提示 SSL 证书错误：执行 git config --global http.sslVerify false 临时关闭验证。
  3. 博客访问 404：检查仓库名是否为「用户名.github.io」，分支是否为 main，部署是否成功。
  4. 本地预览正常，线上样式错乱：检查 _config.yml 中 url 配置是否为你的博客网址（如 url: https://xxxx.github.io）。
